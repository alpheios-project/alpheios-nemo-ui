Welcome to the Alpheios Projet's Text Repository

This environment is an evolution of the original Alpheios text repository. It provides access to the Alpheios Project's library of CapiTainS-compliant texts. CapiTainS is suite of tools and guidelines for publishing texts adhering to the Canonical Text Services (CTS) specification.

Most of the texts in this repository were original published by or with the support of the Perseus Digital Libary.
